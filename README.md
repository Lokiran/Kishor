Employee Management System (Java)
This project is a simple Java console application that demonstrates core Object-Oriented Programming (OOP) concepts. Users can add Full-Time or Part-Time employees and view salary details, bonuses, and additional information like department, experience, leaves, and annual increments.
Features

Add Full-Time & Part-Time Employees
Calculate Monthly & Annual Salary
Bonus Calculation
Annual Increment Projection
Track Department, Experience & Leaves
Uses OOP concepts: Abstraction, Inheritance, Polymorphism & Interfaces
